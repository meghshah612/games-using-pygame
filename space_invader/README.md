# Space Invader

A simple Pygame Space Invader-style shooter.

## Run the game

- Directly: `python main.py`

## Project structure

- `main.py` - top-level entrypoint
- `game.py` - game loop and menu handling
- `entities.py` - ship and laser classes
- `settings.py` - window settings, assets, colors, fonts
- `utils.py` - shared utility helpers
- `assets/` - image assets

## Requirements

- Python 3.x
- Pygame

Install Pygame with:

```bash
pip install pygame
```
