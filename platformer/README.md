# Platformer (Tutorial)

Run the tutorial:

- Using the module: `python -m platformer`
- Directly: `python tutorial.py`

Files:

- `tutorial.py` - platformer runner (wires modules)
- `assets.py` - background, sprites and tile loaders
- `entities.py` - `Player`, `Object`, `Block`, `Fire` classes
- `ui.py` - drawing and collision helpers
- `assets/` - background, sprites and tiles
