# Pong

Simple Pong clone built with Pygame.

Requirements

- Python 3.8+
- Pygame

Install dependencies:

```bash
pip install pygame
```

Run

- Run directly: `python main.py`

Controls

- Left paddle: `W` (up), `S` (down)
- Right paddle: `Up Arrow` (up), `Down Arrow` (down)

Gameplay

- First to 3 points wins (configurable in `main.py`).

Files

- `main.py` - game logic and entrypoint
